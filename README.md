모르는 단어가 생겼을 때 이를 해석과 함께 입력 하고

---

- Sql

## 주요 기능
- 로그인 후 원하는 단어를 저장
  - 하루에 12개 까지 단어 저장 가능
  - 뜻을 잘못 불러오면 자체 수정 가능
  - 원치않는 단어 안보이게 설정(예정)
- 퀴즈 만들기
  - 자신이 저장한 단어들 중 10개를 불러와서 퀴즈 생성
  - 과거 퀴즈 기록 확인 가능
